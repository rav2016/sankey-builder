# Sankey Builder Roadmap

## Near-term
- Node colors (node-level)
- Import/export node layout (`nodes.csv`) enhancements (templates, validation)
- Better auto-level assignment and cycle detection
- UI tooltips and inline help

## Medium-term
- Dark mode and UI polish
- Templates (starter datasets)
- Performance improvements for large datasets

## Long-term
- Desktop packaging (single executable)
- Optional server deployment mode
- Optional persistence / authentication
